# Security Policy

## Supported Versions

The following versions are currently supported:

| Version | Supported |
|--------|-----------|
| 0.x.x  | ✅ Yes    |

## Reporting a Vulnerability

If you discover a security vulnerability in **cosrx/core**, please do **not** open a public GitHub issue.

Instead, report it privately by opening a **GitHub Security Advisory** via the repository’s **Security** tab.

We take security issues seriously and will respond as quickly as possible.
